# erin-reads

> **Reads** — The Field Library
>
> *"It belongs in a museum. Yeah, well, first it belongs in this repo."*

**📜 Live catalog → [erin-reads.pages.dev](https://erin-reads.pages.dev)**

---

Every article worth revisiting. Every thread that had something real in it. Every tool you discovered at 11pm and meant to come back to. Most bookmark apps are graveyards with good search. This one treats every save like an artifact — tagged, annotated, retrievable, and never forgotten.

iOS Share Sheet → shortcut → POST to a Pages Function → a GitHub Issue is born with the URL, OG metadata, and whatever note you left. Close the issue = archived. The archive is still searchable forever.

## Artifacts

- **Reading room** (this, public): the catalog viewer
- **Vault** (private, [`erin-reads-data`](https://github.com/Crosserin/erin-reads-data)): the actual hoard
- **Stack**: Cloudflare Pages + Pages Function + GitHub Issues API

## Clearance

The viewer is public. The collection is not. Choose curiosity wisely.
