## 🗺️ What this is

Every article worth revisiting. Every Hacker News thread that had something real in it. Every tool you discovered at 11pm and meant to come back to. Every *"why wasn't I taught this in school"* post that deserved a second read.

Most bookmark apps are graveyards with good search. **This one treats every save like an artifact** — tagged, annotated, retrievable, never forgotten.

## 🎒 How capture works

```
    📱 iOS Share Sheet
         │
         ↓
    🔗 Shortcut
         │
         ↓
    🌐 POST /reads
         │
         ↓
    ⚡ Pages Function
         │
         ↓
    🐙 GitHub Issues API
         │
         ↓
    ✨ artifact preserved
```

Close an issue = archived. The archive is still **searchable forever**. Nothing gets thrown away, because the whole point is that I *already decided* this was worth keeping.

## 🧰 Stack

- 🔥 Cloudflare Pages + Pages Functions
- 🐙 GitHub Issues API
- 📱 iOS Shortcuts (capture side)
- 🔍 Grep, basically (retrieval side)

## 🗝️ Clearance

The reading room is public. The vault is not. A careful curator is a **private** curator.
